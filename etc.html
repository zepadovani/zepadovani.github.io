harmonics_string.omp (zip)
patch OpenMusic para escrita de harmônicos em instrumentos de corda. input: [1] altura da corda solta (midicents); [2] número do parcial harmônico (int) // output: [1] altura da corda solta (midicents); [2] altura do parcial harmônico resultante (midicents); [3] nós/posições para obter o parcial na corda em questão (midicents)
OpenMusic for writing harmonics in string instruments. input: [1] open string pitch (midicents); [2] harmonic partial number (int) // output: [1] open string pitch (midicents); [2] pitch of the resulting sound (midicents); [3] nodes/positions to obtain the aimed partial in the given string (midicents)
